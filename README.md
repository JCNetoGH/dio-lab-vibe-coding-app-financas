# Contexto
Quero criar um app de Organização de Finanças Pessoais que funcione através de conversa.  
Quero deixar tudo mais simples, sem planilhas ou formulários chatos.

# Problema
A maioria das pessoas desanima de controlar o dinheiro porque os apps são complicados e nada pessoais.  
Quero criar uma experiência por chat, com dicas automáticas de economia.

# Público-Alvo
Quem quer começar a organizar o dinheiro sem complicação.

# Funcionalidades Principais
1. Registrar gastos via chat usando linguagem comum.  
2. A IA organiza tudo automaticamente.  
3. Criar e acompanhar metas.  
4. Ter um “Agente Financeiro” dando dicas.  
5. Mostrar relatórios simples e personalizados.

# O que a IA deve entregar
Um plano de MVP com as principais telas, os recursos necessários e como validar a ideia.  
Linguagem simples e didática.
